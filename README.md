# cicdez

## Overview

Simple CLI to manage deployments, configuration, and secrets. Everything stays encrypted in your repo. Build, push, and deploy to Docker Swarm with one command.

## Installation

```bash
go install github.com/blindlobstar/cicdez/cmd/cicdez@latest
```
